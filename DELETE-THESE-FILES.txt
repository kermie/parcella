Nach dem Entpacken dieses ZIP-Archivs in dein Projektverzeichnis (Dateien
überschreiben) bitte diese beiden alten, jetzt umbenannten Template-Dateien
manuell löschen (git kennt sie noch unter dem alten Namen, das ZIP enthält
nur die neuen Dateien):

  app/templates/tickets/uebersicht.html   (ersetzt durch overview.html)
  app/templates/tickets/formular.html     (ersetzt durch form.html)

Danach:

  git add -A
  git commit -m "refactor: rename ticket module to English (Ticketsystem -> Tickets)"
  git push

Der Commit wurde im Sandbox-Repo bereits mit passender Commit-Message und
korrekter Git-Identität (Marco Steinhaeuser) vorbereitet; du kannst die
Commit-Message oben 1:1 übernehmen oder git commit --amend nutzen, falls du
stattdessen den vorbereiteten Commit direkt einspielen möchtest (dann bräuchtest
du aber den vollständigen Patch statt des ZIPs – sag Bescheid, falls dir das
lieber ist).

Migration nicht vergessen, sobald der Server läuft:
  docker compose exec web alembic upgrade head
