Hinweis: Das Modul-Flag hat sich von "einkaufswuensche" auf
"purchase_requests" geändert. Falls in eurer Vereinseinstellungen-Tabelle
explizit "modul_einkaufswuensche" auf false gesetzt war, greift nach dem
Deploy wieder der Default (true/aktiviert), weil der Code jetzt nach
"modul_purchase_requests" sucht. Das war bei den vorherigen Modul-Umbenennungen
(Versicherungen -> Insurance, etc.) genauso und wurde bisher nicht per
Migration nachgezogen – falls euch das wichtig ist, sag Bescheid, dann
ergänze ich das nachträglich für alle betroffenen Module in einer
